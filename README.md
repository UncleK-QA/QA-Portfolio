# QA Portfolio

This repository contains my QA projects, including:

- Playwright automation tests for login forms, Google integration, and website navigation
- Manual test cases and sample bug reports
- Project configuration (`playwright.config.js` and `package.json`)

## Skills demonstrated

- Automated testing with Playwright
- JavaScript programming
- Writing clean and maintainable test cases
- Manual QA documentation

## How to run the tests

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npx playwright test` to execute the tests
